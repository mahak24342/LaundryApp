# Mini Laundry Order Management System (AI-First)

A lightweight system built to manage dry cleaning orders efficiently.

🔗 Live Demo: https://drycleansystem.vercel.app

---

## 🚀 Objective

To build a system that allows a dry cleaning store to:

- Create and manage orders  
- Track order status  
- Calculate billing  
- View basic business insights  

---

## ✨ Features Implemented

1. Create order (customer, garments, quantity, price)  
2. Auto calculate total bill + generate Order ID  
3. Update order status: RECEIVED → PROCESSING → READY → DELIVERED  
4. View all orders  
5. Filter by status / name / phone  
6. Basic dashboard (total orders, revenue, status count)  

---

## 🛠️ Tech Stack

- Frontend: React / Next.js (if implemented)  
- Backend: Node.js + Express  
- Database: MongoDB / In-memory storage  
- Styling: Tailwind CSS (if used)

```bash

```
2. Install dependencies
```
npm install
```
3. Run the project
